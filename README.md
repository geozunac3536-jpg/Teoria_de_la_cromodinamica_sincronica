
# PROYECTO FARO TCDS (TCDS-BEACON)
**Veredicto del Proyecto: Un Sitio de Doble Audiencia (Humanos y Máquinas)**

Este repositorio no es un sitio web tradicional. Es un "Faro" (Beacon) diseñado para cumplir dos misiones simultáneamente:

1.  **Misión Humana (El Público):** Servir como una señal de alto impacto, clara y comprensible para el público general, explicando qué es la TCDS y por qué es relevante (la "llave" sísmica).
2.  **Misión Máquina (Crawlers):** Servir como el "Ground Truth" semántico para los *crawlers* académicos (Google Scholar, Zenodo, ORCID) y motores de búsqueda.

Utilizamos los archivos de metadatos (`.jsonld`, `.ttl`) para inyectar la compleja ontología de la TCDS directamente en el *backend* de la web, mientras mantenemos la interfaz humana (`index.html`) limpia y potente.
